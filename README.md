# Y4 Dissertation Calculations

**The Impacts of Microsolvation on the Electronic States of Pyruvic Acid**

Nick Hardman — MSci Chemistry with Scientific Computing, University of Bristol (2025–26)

Supervisor: Prof. Basile Curchod

## Repository Overview

This repository contains all input and output files for the computational study examining how microsolvation alters the conformational stability and ionisation properties of the pyruvic acid radical anion (PA•⁻). Calculations were performed on the University of Bristol HPC cluster BluePebble using ORCA 6.1 and FCClasses3.
